Atenção para rodar o backend

É necessário:
    - baixar o node
    - criar uma conta no mongodb e usar o mongo atlas para rodar, é direto na web
    - Assim que criar uma conta, seguir os passos desse tutorial a partir do "Creating an account for MongoDB Atlas"
    https://blog.logrocket.com/mern-stack-tutorial/ até o texto "Our database is ready"
    O nome do banco é Cluster0, usuario e senha é gerencia. muito criativo
    - Eu vou ver se tem como a gente compartilhar o que eu criei aqui, senão vão ter q fazer isso mesmo

Para rodar:
    Entrar na pasta backend pelo cmd, usar "npm install" que vai baixar as dependencias para rodar.
    assim que terminar, "node server.js"
